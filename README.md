# Nova2SonicTestHarness
End-to-end test harness for Amazon Nova 2 Sonic. Simulates multi-turn conversations via Bedrock bidirectional streaming with configurable LLM user simulators, scripted inputs, Polly TTS audio, tool-use scenarios, session continuation, and binary LLM-as-judge evaluation. Supports batch runs, dataset-driven testing, and a Streamlit dashboard.
